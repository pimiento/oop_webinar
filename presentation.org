* super
  $super()$ возвращает объект родителя, чтобы мы могли запросить нужный нам метод/атрибут у родителя

    class B(A):
        def function(self):
            super().method()
            (
                self.__class__
                .__bases__[0]
                .method(self)
            )
            print("function")
  #+END_SRC
